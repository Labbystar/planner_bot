# Planner Bot

Telegram planner bot with:
- кнопочным меню
- несколькими получателями
- таймзонами
- share-ссылками
- категориями и приоритетами
- историей действий
- кнопками «выполнено / отложить»

## Railway
Start command:

```bash
python run.py
```

Variables:

```env
BOT_TOKEN=...
BOT_USERNAME=your_bot_username
DEFAULT_TIMEZONE=Asia/Novosibirsk
```
